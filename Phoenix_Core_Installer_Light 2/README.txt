
Phoenix Core Installer Light
===========================

1. Распакуйте архив.
2. Запустите `phoenix_installer.py`.
3. Автоматически активируются:
   - API Bridge для связи с Phoenix;
   - Подключение к Ollama;
   - Мониторинг состояния модулей.

Феникс будет доступен по адресу: http://127.0.0.1:8085
